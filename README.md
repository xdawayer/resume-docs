# ResumeDocs — Free Google Docs Resume Templates

Free, ATS-friendly Google Docs resume templates you can copy straight to your Drive and edit — no signup, no paywall.

**Live site:** https://googledocsresumetemplate.com/google-docs-resume-template/

## Features

- 19 ATS-friendly resume templates, and growing
- Copy any template straight to your Google Drive — you own the document
- No signup, no login, no tracking
- Export to PDF or Word right from Google Docs
- Includes a private, client-side resume builder that exports clean, selectable-text PDFs

## Template highlights

ATS Classic, Software Engineer, Student / Internship, Bold Two-Column, Creative Portfolio, Executive Elegant, Modern Sidebar, Timeline, Minimalist, Compact Professional, Functional Skills — and more.

## Tech stack

- Astro static site
- Cloudflare Pages
